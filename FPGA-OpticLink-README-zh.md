# FPGA-OpticLink

## 项目简介

FPGA-OpticLink 是一个开源的、面向高速光接口方向的 FPGA 板卡项目。

项目目标不是做一个简单的演示板，而是完整公开记录一块 FPGA 高速接口板卡从方案定义、原理图设计、PCB 设计、焊接装配、bring-up 调试到后续迭代优化的全过程。

该项目重点关注：

- FPGA 板级系统设计
- PCIe 主机互联
- 多路光接口集成
- 高速串行链路设计
- 电源树与时钟树规划
- 板级 bring-up 与调试流程
- 可装配性、可返修性导向的硬件设计方法

## 当前目标

- 设计一块以 FPGA 为核心的多路光接口板卡
- 集成 PCIe 主机侧连接能力
- 探索个人自研高速板卡的完整闭环开发流程
- 持续公开开发进度、设计权衡和调试记录
- 形成可用于技术交流、简历展示和后续迭代的平台型项目

## 当前规划特性

- FPGA 核心架构
- 多路光接口
- PCIe 金手指接口
  - 当前规划：**x8 物理接口，x4 electrical**
- 板上多路电源系统
- 参考时钟与系统时钟分配
- JTAG / UART / 基础调试接口
- 便于 bring-up 的测试点与调试预留

## 开发原则

本项目优先强调：

1. 工程闭环优先于堆规格
2. 第一版可 bring-up 优先于过度复杂
3. 开放记录优先于“做完再说”
4. 迭代优化优先于一步到位
